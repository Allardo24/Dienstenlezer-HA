# DienstenLezer voor Home Assistant

Deze Home Assistant-app draait de bestaande DienstenLezer-webinterface en Rust-backend als een ARM64- of AMD64-container. De interface en functies blijven gelijk aan de normale webversie.

Ondersteund:

- Raspberry Pi 4 met 64-bits Home Assistant OS (`aarch64`)
- Home Assistant OS op `amd64`
- Blijvende opslag in `/data`
- Gezondheidscontrole via `/api/health`
- Automatisch starten met Home Assistant
